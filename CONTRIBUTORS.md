## 如何成为贡献者

1. 使用「**预设构建器**」制作预设，填写作者名
2. 打开「**官方预设上传工具**」，填入你的 GitHub 用户名
3. 选择预设文件 → 提交投稿
4. 等待审核合并后，你的名字会自动出现在这里
